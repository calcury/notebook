# 无监督学习
## 聚类
### 1. K-means
- 寻找就近点聚集
- 结果只能接近球形
### 2. DBSCAN
- 按密度搜索
- 结果可以是任意形状，无需提前制定簇的数量
### 3. HDBSCAN
- 按密度分层检索
- 可以根据不同层次查询不同聚类
## 降维
### 1. PCA
- 计算协方差矩阵，求特征向量，按特征值从大到小排序，将数据投影到特征值较大的维度
- 降维 + 去相关性
- 速度快，可解释性强
### 2. t-NSE
- 保留局部结构，可能导致高维相关性丢失
- 局部可视化聚类最强
### 3. UMAP
- 同时保留整体和局部结构

# 模型评估
### 1. Within-cluster sum of squares (WCSS) / Inertia

- 所有样本到其所属簇中心的距离平方和
  $$\text{WCSS} = \sum_{i=1}^K \sum_{x \in C_i} \|x - \mu_i\|^2$$
  其中 $K$ 是簇数，$C_i$ 是第 $i$ 个簇，$\mu_i$ 是该簇的中心。
  - 横轴：簇的数量 $K$
  - 纵轴：WCSS 值
  - 随着 $K$ 增大，WCSS 会单调递减，因为簇越多，每个簇越紧。
  - Elbow：曲线下降速度突然变缓的点，这个点对应的 $K$ 通常是最优簇数
- 特点：
  - 计算简单，是 K-Means 的目标函数
  - 对异常值敏感，因为平方会放大远距离点的影响
  - 不考虑簇间分离度，只关注簇内紧凑性

---

### 2. Silhouette Score
是综合评估簇内紧凑性和簇间分离度的指标

- 对每个样本 $i$，计算其轮廓系数 $s(i)$，再取所有样本的平均值。
  $$
  s(i) = \frac{b(i) - a(i)}{\max(a(i), b(i))}
  $$
  - $a(i)$：样本 $i$ 到同簇内其他样本的平均距离（内聚度）
  - $b(i)$：样本 $i$ 到最近异簇内所有样本的平均距离（分离度）
  - 横轴：簇的数量 $K$
  - 纵轴：平均轮廓系数
  - 取值范围 $[-1, 1]$：
    - 接近 1：样本聚类合理，簇内紧、簇间远
    - 接近 0：样本位于簇边界，归属模糊
    - 小于 0：样本更可能属于其他簇
  - 找“峰值”：曲线的最高点对应的 $K$ 通常是最优簇数
- 特点:
  - 比 WCSS 更全面，同时考虑了簇内和簇间
  - 计算成本较高，不适合超大数据集
  - 对球形簇效果好，对非凸簇或密度差异大的簇效果不佳

---

### 3. Davies-Bouldin Index (DBI)
是衡量聚类“紧凑且分离”程度的指标

- 计算每一对簇的“相似度”，取平均值，相似度定义为两个簇的平均距离之和除以它们中心之间的距离
  $$
  \text{DBI} = \frac{1}{K} \sum_{i=1}^K \max_{j \neq i} \left( \frac{\sigma_i + \sigma_j}{d(c_i, c_j)} \right)
  $$
  - $\sigma_i$：簇 $i$ 内样本到中心的平均距离
  - $d(c_i, c_j)$：簇 $i$ 和簇 $j$ 中心之间的距离
  - 横轴：簇的数量 $K$
  - 纵轴：DBI 值
  - DBI 越小，聚类效果越好，说明簇内更紧凑、簇间更分离
  - 找“谷值”：曲线的最低点对应的 $K$ 通常是最优簇数
- 特点：
  - 计算简单，对簇的形状不敏感
  - 对异常值敏感，因为平均距离会被拉远
  - 不适合评估包含噪声点的聚类
